Release Notes

11/08/2005

General

  • Enhancement: Unique keys are now defaulted to Auto-Assign.

Strain Edit

  • Bug: Strain was not being added correctly to database.

Tumor Edit

  • Bug: Tumor was not being added correctly to database.
  • Bug: Tumor information was not being added to database.
  • Enhancement: Tumor Frequency Treatments tab now has an Agent filter for easier searching of the combo box.
  • Enhancement: Combo boxes now "scroll" to an item when a key is pressed.

10/26/2005

General

  • Bug: Some fields were not displaying the full text. Adjusted the size to be bigger.
  • Enhancement: JNumber entry fields now automatically parse and correct short hand versions of JNumbers.
  • Enhancement: Can now double click on a row in the search results table to launch the edit window.
  • Enhancement: The hover text only appears for the necessary columns.
  • Enhancement: The search results columns size is more appropriate to the data being displayed.
  • Enhancement: The search pick lists now handle keyboard events to make scrolling of long lists easier to navigate.

Tumor Search

  • Bug: Search not handling all parameters correctly. Fixed the search pass all parameters were to the backend search.
  • Bug: The reset button on the Tumor search form doesn't reset the checkboxes. Fixed to reset checkboxes.
  • Enhancement: The JNumber and Reference fields were not clear on what was being asked for. The JNumber is the actual JNumber used (ie, J:46464). The only time a "key" is used is where it is specifically stated "key" and when you see the key icon.

Strain Edit

  • Enhancement: Strain References now display the JNumber and the short citation.
  • Enhancement: The Strain Type section has been redeveloped to show 2 lists, one for available and one for selected. Drag and Drop is also supported between the lists.

Tumor Edit

  • Bug: Progression information was not displaying. It now is.
  • Bug: When exiting form, it erroneously reported that you changed a value when you did not. It now works correctly.
  • Enhancement: Changing the associated Strain now gives visual clue to it being changed.
